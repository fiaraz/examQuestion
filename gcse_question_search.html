AQA GCSE Question Search — prototype

Search past-paper questions by topic. A match on any part of a question (e.g. 04.2) returns the whole question (04.1–04.n) so you get the full context, not just the fragment that matched.

Diagrams: these papers were parsed as text, not images, so a real diagram can't be pulled out of the PDF automatically. Any part mentioning a Figure/Table is flagged with a badge and a link to the original AQA PDF where you can see it.
